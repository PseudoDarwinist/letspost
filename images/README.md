# Images Folder

## How to Add Images

### Background Images
Save your background images here and name them:
- background-001.jpg (retro waves - warm tones)
- background-002.jpg (cyberpunk city - teal tones)
- background-003.jpg
- etc.

### Diagram Images
Save your architecture diagrams here:
- diagram-001.png
- diagram-002.png
- etc.

### Supported Formats
- JPG/JPEG
- PNG
- GIF
- WebP
